# Use Case

## 목적
구체적인 비즈니스 시나리오(유스케이스)를 구현합니다.

## 특징
- "사용자가 ~한다"로 표현되는 기능
- 하나의 유스케이스 = 하나의 클래스
- 트랜잭션 경계
- 여러 리포지토리와 도메인 객체 조율

## 포함될 파일
- `send_message.py`: 메시지 전송
- `join_room.py`: 채팅방 입장
- `leave_room.py`: 채팅방 퇴장
- `create_room.py`: 채팅방 생성
- `get_message_history.py`: 메시지 히스토리 조회

## 책임
- 비즈니스 시나리오 완전 구현
- 도메인 객체와 리포지토리 조율
- 트랜잭션 관리
